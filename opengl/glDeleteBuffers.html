glDeleteBuffers

删除指定的缓冲区对象。

C规范

void glDeleteBuffers(GLsizei n,
　　　　　　const GLuint * buffers);

参数

n

指定将要被删除的缓冲区对象的数量。

buffers

指定将要被删除的缓冲区对象的一个序列。

描述

glDeleteBuffers 删除由buffers数组元素指定的n个缓冲区对象。在一个缓冲区对象被删除以后，它将不再有任何内容，并且它的名字已经可以进行重用了（例如被glGenBuffers重用）。如果一个当前被绑定的缓冲区对象被删除，绑定（binding）将回复到0（任何缓冲区对象的缺失）。

glDeleteBuffers 默认忽略0和不符合现有缓冲区对象的名称。

错误

如果n为负值，则产生GL_INVALID_VALUE 错误。
