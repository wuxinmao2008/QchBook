glDeleteVertexArrays

删除顶点数组对象。

C规范

void glDeleteVertexArrays(GLsizei n,
　　　　　　const GLuint * arrays);

参数

n

指定将要被删除的顶点数组对象的数量。

arrays

指定一个包含将要被删除的n个对象名称的数组的地址。

描述

glDeleteVertexArrays删除n个顶点数组对象，这些顶点数组对象的名称被存储在由arrays进行寻址的数组中。一旦一个顶点数组对象被删除，它就不再包含任何内容，并且它的名称就会再次变为未使用的。如果一个当前被绑定的顶点数组对象被删除，那么这个对象的绑定将回复到0，并且默认顶点数组将会变成当前顶点数组。arrays中未使用的名称将被默认忽略，就像zero值一样。

错误

如果n为负值，则产生GL_INVALID_VALUE 错误。

glGenVertexArrays, glIsVertexArray, glBindVertexArray
